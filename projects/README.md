# My Portfolio Website

## Welcome to my website!
Welcome to my personal portfolio project website! Built with HTML, CSS and Jekyll,
this website is designed as an interactive CV, where recruiters can view
my languages, skills, toolsets and details about possible projects when at MVP level.

Below are the features of the website:

### Home Page
Within this page we have a number of features such as:
- Language Wrapper Carousel
- Education Tile Card
- Project Nav Buttons
- UX Features (Programming Language Carousel, Education Card Flipping, Scroll to Top Button, Navigation Buttons to Page Sections)
  
### Individual Project Pages (DocuLift)
Within this page we have the following features:
- Demo Video Display
- Collapsible Elements that answer project FAQs
- Tech Stack Described

### Small Projects
Within this page we have the following features
- Language Sections
- Individual Nav Buttons to each individual repository - tiled for enhanced UX/viewing
